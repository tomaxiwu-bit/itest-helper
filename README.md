## 安装

### 准备工作

安装脚本管理器（任选其一）：

| 管理器 | 适用浏览器 | 推荐 |
|--------|-----------|------|
| [ScriptCat (脚本猫)](https://docs.scriptcat.org/) | Edge / Chrome | ⭐ 国产，功能丰富 |
| [Tampermonkey (篡改猴)](https://www.tampermonkey.net/) | Edge / Chrome / Firefox / Safari | ⭐ 最通用 |
| [Violentmonkey (暴力猴)](https://violentmonkey.github.io/) | Edge / Chrome / Firefox | 开源轻量 |

### 安装脚本

**方法一：直接安装（推荐）**

👉 [点击此链接安装 itest-helper.user.js](itest-helper.user.js)

脚本管理器会自动弹出安装提示，点击「安装」即可。

**方法二：手动安装**

1. 打开 itest-helper.user.js 文件，复制全部代码
2. 点击浏览器右上角脚本管理器图标 → 「新建脚本」
3. 粘贴代码 → Ctrl + S 保存

## 界面说明

安装后进入 iTest 考试页面，右上角会出现操作面板：

`
┌─────────────────────────┐
│  iTest Helper        ✕  │  ← 标题栏（可拖拽）
├─────────────────────────┤
│                         │
│  ┌─────────────────┐   │
│  │ 粘贴答案 JSON... │   │  ← 答案输入区
│  └─────────────────┘   │
│                         │
│  [扫描题目]  [保存答案]  │  ← 辅助按钮
│                         │
│  [  填写当前页  ] [全部] │  ← 核心功能按钮
│                         │
│  ┌─────────────────┐   │
│  │ 运行日志...      │   │  ← 状态输出区
│  └─────────────────┘   │
└─────────────────────────┘
`

### 各按钮作用

| 按钮 | 作用 |
|------|------|
| **扫描题目** | 扫描当前页面有多少道题，确认题数是否与答案匹配 |
| **保存答案** | 将答案保存到本地，下次打开页面自动加载 |
| **填写当前页** | 只填写当前页面的题目 |
| **全部** | 从当前页开始，逐页填写并自动翻页，直到最后一页 |

## 使用步骤

### 首次使用

1. 进入 iTest 考试页面（如 itestcloud.unipus.cn/itest-api/itest/s/answer/index?token=...）
2. 点击 **扫描题目**，查看当前页有多少道题
3. 将你的答案以 JSON 格式粘贴到输入框中
4. 点击 **保存答案**（下次自动加载，无需重复粘贴）
5. 点击 **填写当前页** 验证答案是否正确匹配
6. 确认无误后点击 **全部**，脚本会自动翻页填写所有题目

### 再次使用

再次打开考试页面时，之前保存的答案会自动加载，直接点击 **全部** 即可。

## 答案格式

### 标准格式

`json
["A","B","C","D","A","B","C","D"]
`

### 也支持对象格式

`json
{"answers": ["A","B","C","D"]}
`

### 示例

假设第 1 题选 A，第 2 题选 B，第 3 题选 C，共 200 题：

`json
["A","B","C","A","B","C","A","B","C","A","B","C","A","B","C"]
`

> ⚠️ 答案数量必须与页面上扫描到的题目数量一致，否则末尾题目会留空。

## 映射规则

脚本按题目在页面上出现的 DOM 顺序逐题匹配：

| 答案 | 对应 |
|------|------|
| A | 第 1 个选项 (value=0) |
| B | 第 2 个选项 (value=1) |
| C | 第 3 个选项 (value=2) |
| D | 第 4 个选项 (value=3) |

> 写作/填空题会自动填入对应文本。

## 常见问题

### Q: 脚本没反应？

- 确认已安装 ScriptCat 或 Tampermonkey 并已启用
- 确认页面 URL 以 itestcloud.unipus.cn/itest-api/itest/s/answer/index 开头
- 刷新页面重试

### Q: 答案数量和题目数量对不上？

- 点击 **扫描题目** 查看实际题数
- 调整答案数量后重试
- 如果页面上有多余的非答题元素（如听力音频），脚本会自动跳过

### Q: 翻页后不填写？

- 确保答案数量足够覆盖所有页的题目
- 翻页间隔为 2.5 秒，请耐心等待

### Q: 填写错了怎么办？

- 刷新页面即可重置所有选项
- 答案修正后重新点击填写

## 自定义设置

### 修改翻页上限

默认最多翻 200 页，如需修改，在浏览器控制台 (F12) 输入：

`javascript
GM_setValue("itest_maxPages", 999)
`

### 清除已保存的答案

`javascript
GM_setValue("my_answers", "")
`
